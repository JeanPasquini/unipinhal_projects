TAREFA

Descrição:

Fazer um algoritmo universal que represente o Autômato Finito Determinístico

///

CONFIGURAÇÃO DO AMBIENTE DE DESENVOLVIMENTO 

Linguagem Utilizada: Python

IDE: Visual Studio Code

///

PROJETO CONCLUÍDO

Introdução:

O algoritmo tem por função proporcionar qualquer Autônomo Finito Determinístico com base nas informações dadas pelo usuário.

Uso:

Programa pergunta ao usuário:

(Q, Σ, δ, q0, F)

Conjunto de estados, Alfabeto, Transições, Estado inicial e Conjunto de estados finais respectivamente.

Após isso o sistema pergunta a entrada que o usuário deseja testar.

Dado essas informações são armazenadas em variáveis para a classe AFD (classe criada para resolver o algoritmo por meio de operações lógicas),
logo em seguida é informado se a entrada fornecida pelo usuário é possível realizar ou não.
